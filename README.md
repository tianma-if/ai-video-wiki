# 🎬 AI 视频制作速查手册 (AI Video Wiki)

这是一个**纯开源、高密度、极简黑白风格**的 AI 视频制作提示词与工作流速查工具。

本项目基于现代内容框架 Astro v5 与 MDX 构建，并采用 Bun 作为运行时。这种技术架构确保了整站静态直出、页面毫秒级秒开以及极致的搜索引擎（SEO）收录友好度。

---

## 💡 我们能为你做什么？（核心功能）

该网站不是冗长的教程，而是一个专为 AI 视频创作者设计的**“词条字典”**。普通用户和创作者可以用它实现以下目标：

*   **🎬 运镜效果直观预览**：文字无法表达镜头运动？只需将鼠标悬停在“推镜头”、“环绕运镜”等卡片上，即可瞬间循环播放对应的运镜短片预览，快速理解镜头效果。
*   **📋 提示词一键复制**：网站收录了 Kling、Runway、Luma、Midjourney 等各大主流 AI 模型的最优控制指令。每个指令旁边都配有一键复制按钮，省去手动打字和记忆提示词的烦恼，即拿即用。
*   **📐 轻松控制分镜与景别**：不知道如何写出特写、俯瞰、荷兰斜角？在这里可以直接查阅标准的景别修饰词，直接输入给 AI 视频或画图工具，即可完美锁定构图，避免画面崩坏。
*   **📝 快速生成爆款脚本**：提供爆款视频黄金前3秒开头公式、情感共鸣驱动以及经典“三幕式”、“英雄之旅”叙事脚本提示词，粘贴进 Claude/ChatGPT 即可快速产出高质量视频大纲。
*   **🎨 解决后期与配音难题**：速查“面部闪烁修复”、“视频无损放大 4K”以及“电影级背景音乐 BGM 生成标签”等成熟的后期优化方案。
*   **🤝 极其简单的开源共建**：所有页面都是 Markdown 文件。如果你发现了更好用的 Prompt，可以直接编辑项目文件提交贡献，与全球创作者共享。

## 🚀 快速开始

### 1. 安装
```bash
bun install
```

### 2. 本地预览
```bash
bun run dev
```
启动后在浏览器中访问 `http://localhost:7171`。

### 3. 静态打包
```bash
bun run build
```
